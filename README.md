# TASK-6-SALES-TREND-ANALYSIS
SQL Project analyzing monthly revenue and order volume using aggregate functions, GROUP BY, and JOINS in the Northwind database.

Project Overview

This project analyzes monthly sales trends using the Northwind database in SQL Server Management Studio (SSMS). The analysis focuses on calculating monthly revenue and order volume using SQL aggregate functions.

Objectives

Analyze monthly revenue.

Analyze monthly order volume.

Practice SQL aggregation functions.

Group data by year and month.

Sort results in chronological order.


Tools Used

SQL Server Management Studio (SSMS)

Northwind Database

SQL


SQL Concepts Used

INNER JOIN

SUM()

COUNT(DISTINCT)

GROUP BY

ORDER BY

YEAR()

MONTH()


Revenue Formula

Revenue was calculated using the formula:

UnitPrice * Quantity * (1 - Discount)

Outcome

The query successfully generated a monthly sales trend report showing:

Monthly revenue

Monthly order volume


This analysis demonstrates how SQL aggregate functions can be used to summarize sales performance and identify trends over time.
